# Logos de Convenios

Esta carpeta contiene los logos de las empresas con convenio.

## Logos necesarios:

1. **ispeakcenter.png** - Logo de Ispeakcenter (Escuela de inglés)
2. **bares.png** - Logo de Bares
3. **alimentos-saludables.png** - Logo de Alimentos Saludables
4. **centro-deportivo.png** - Logo del Centro Deportivo
5. **welly-protein.png** - Logo de Welly Protein

## Especificaciones:

- Formato recomendado: PNG con fondo transparente
- Tamaño recomendado: 300x300px o mayor (ratio 1:1)
- Los logos se mostrarán dentro de círculos blancos con borde dorado
- Asegúrate de que los logos tengan buena resolución para verse bien en pantallas de alta densidad
